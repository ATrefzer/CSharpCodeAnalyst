1. ELKJS (elk.bundled.js)
================================================================================

https://github.com/kieler/elkjs

elkjs is a JavaScript port of the Eclipse Layout Kernel (ELK). It computes the
node positions for the "layered" graph layouts offered by the web graph view.
Bundled locally under
CSharpCodeAnalyst/Features/WebGraph/Web/lib/elk.bundled.js.
